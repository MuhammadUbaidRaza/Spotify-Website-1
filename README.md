🎧 Spotify Frontend Website Description

This project is a visually appealing frontend clone of the Spotify web player, built using HTML and CSS. It replicates the modern and sleek user interface of Spotify, focusing on layout design, responsiveness, and user experience.

The website features a dark-themed UI with a sidebar for navigation, including sections like Home, Search, and Your Library. The main content area showcases playlists, albums, and artists in a grid-based layout with interactive hover effects.

A fixed music player bar is designed at the bottom of the page, displaying song details, playback controls, and progress indicators. The design emphasizes clean typography, smooth spacing, and consistent color schemes to closely match Spotify’s aesthetic.

Key frontend features include:

Responsive layout using Flexbox and CSS Grid
Sidebar navigation menu
Playlist and album cards with hover animations
Sticky footer music player UI
Modern dark theme styling

This project is ideal for practicing UI design, layout structuring, and CSS styling techniques, without backend functionality.
